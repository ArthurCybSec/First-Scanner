## Descrição
Scanner simples de portas TCP desenvolvido em Python com fins educacionais,
voltado para o aprendizado de redes, sockets e fundamentos de segurança.

## Objetivo
Praticar conceitos básicos de:
- Comunicação TCP/IP
- Uso da biblioteca socket
- Reconhecimento inicial em segurança

## Como executar
python src/port_scanner.py

## Tecnologias
- Python 3
- Biblioteca socket

## Limitações
- Scanner sequencial
- Lista de portas reduzida
- Não substitui ferramentas profissionais como Nmap
